大樂透

2023/5/30 程式筆記 大樂透

抽籤/取亂數

　　　大樂透最核心的部分其實就是產生一組兌獎號碼，雖然我沒辦法寫出一模一樣的程式，但是我可以簡單的模擬一下。

目標

　　　抽出十期對獎號碼，且使用者可以選擇期別和號碼查詢有無中獎，最後再詢問要不要繼續查詢

程式碼

#include <iostream>
#include <time.h>
#include <stdlib.h> 

using namespace std;

int main()
{
    int result[10][7];
    
    for(int i = 0; i < 10; i++) {
        for(int j = 0; j < 7; j++) {
            result[i][j] = 0;
        }
    }
    
    for(int i = 0; i < 10; i++) {
        for(int j = 0; j < 7; j++) {
            result[i][j] = rand() % 50 + 1;    //取亂數
            
            for(int k = 0; k < j; k++) {              //不重複亂數
                if(result[i][j] == result[i][k]) {   
                    j--;
                    break;
                }
            }
        }
    }         
    
    for(int i = 0; i < 10; i++) {
        for (int j = 0; j < 5; j++) {  
            for (int k = 0; k < 5-j; k++) {
                if(result[i][k] > result[i][k+1]) swap(result[i][k], result[i][k+1]);        //泡沫排序    
            }
        }
    }
    
    for(int i = 0; i < 10; i++) {
        cout << "第 " << i+1 << " 期中獎號碼:";
        for(int j = 0; j < 6; j++) {
            cout << result[i][j] << " \t";
        }
        cout << "特別號: " << result[i][6] << endl;  //列出每一期號碼
        
    }
    char yn = 'y';
    while(yn == 'y' || yn == 'Y') {         //如果輸入y或Y就繼續詢問
        cout << "\n請問要查詢第幾期\n";
        
        int w,number;     //輸入期別 號碼
        cin >> w;
        
        cout << "請問要查詢什麼號碼\n";
        cin >> number;
        
        for(int i = 0; i < 7; i++) {               //判斷有沒有中獎
            if(result[w-1][i] == number) { 
                cout << "第" << w << "期中獎號碼/ " << number << ":中獎\n是否要繼續(Y/N)"; break;}
            else {
                cout << "第" << w << "期中獎號碼/ " << number << ":未中獎\n是否要繼續(Y/N)"; break;}
        }
        
        cin >> yn;          //如果輸入的字不對就重新輸入
        while(yn != 'y' && yn != 'Y' && yn != 'n' && yn != 'N') {
            cout << "請重新輸入(Y/N)\n";
            cin >> yn;
        }
    }
}

運作原理

  1. 宣告一個大小為 10 * 7 的二維陣列，用來儲存每期七個號碼共十期
  2. 把時間當作rand()函式的亂數種子，生成每期不重複亂數共十期 範圍:1~50
  3. 利用泡沫排序法把每期的前六個數字由小到大排序，第七個數字當作特別號
  4. 列出每一期的號碼
  5. 輸入欲查詢的期別與號碼，如果文字輸入錯誤要求重新輸入，大小寫皆可
